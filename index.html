Features

機能

🎹

Mechanical Keyboard Sounds

メカニカルキーボード音

Pleasant typing sounds that mimic high-quality mechanical keyboards.

高品質なメカニカルキーボードを模した心地よいタイピング音。

🔊

Special Key Sounds

特殊キーサウンド

Custom sounds for Enter, Tab, Space, and Backspace keys.

エンター、タブ、スペース、バックスペースキーに独自の効果音。

🔄

Anti-Key-Repeat

キーリピート対策

Prevents sound flooding when keys are held down.

キーを長押ししたときの音の洪水を防止。

🔧

Customizable

カスタマイズ可能

Adjust volume and enable/disable specific key sounds.

音量調整や特定のキー音のオン/オフ切り替え。

How to Use

使い方

1

Install the Extension

拡張機能をインストール

Install "VSCode Typing Sound" from the VS Code Marketplace.

VS Code マーケットプレイスから「VSCode Typing Sound」をインストールします。

2

Start Coding

コーディングを開始

The sounds will play automatically as you type.

タイピングに合わせて自動的に音が再生されます。

3

Customize Settings

設定をカスタマイズ

Adjust volume and other settings in VSCode preferences.

VSCodeの設定で音量やその他の設定を調整します。

Settings

設定オプション

Setting設定
Description説明
typingSound.enabled
Enable or disable typing sounds 効果音を有効/無効にする
typingSound.volume
Sound volume (0-100) 音量（0-100）
typingSound.keyCooldown
Minimum interval between sounds for the same key (milliseconds) 同一キーの再生間隔（ミリ秒）
typingSound.specialKeys
Enable/disable specific special key sounds 特殊キーの有効/無効設定
